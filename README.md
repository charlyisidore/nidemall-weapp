# NideMall (WeChat mini program)

WeChat mini program for [NideMall](https://github.com/charlyisidore/nidemall-server).

## Installation

Create or edit `project.private.config.json`:

```json
{
  "appid": "touristappid",
  "projectname": "nidemall-weapp",
  "setting": {
    "compileHotReLoad": true
  }
}
```
